Multi-Sequence Input

T1
T1ce
T2
FLAIR

All four sequences fused into a single 3D volumetric analysis — the way expert neuroradiologists read.

  • Sliding-window 3D inference — analyzes the full brain volume, not just 2D slices, ensuring no region is missed.
  • Isotropic resampling to 1 mm — normalizes for scanner variability across field strengths and vendors.
  • Z-score intensity normalization — robust to differences in contrast agent dosing and scanner calibration.
  • Volume in milliliters (mL) — per sub-region, plus percentage contribution, ready for response assessment (RANO criteria).
  • Model lineage tracking — every result records model version + SHA-256 checksum for full reproducibility and regulatory audit trails.
  • Minimum lesion filter (0.05 mL) — suppresses spurious sub-voxel detections while preserving clinically relevant findings.
Quality Assurance

Built-In Image Quality Guard

The platform automatically evaluates image quality before AI inference — so radiologists are never handed a result based on degraded data.

🎯

Missing Sequence Detection

Flags studies where required MRI sequences (T1, FLAIR, T2) are absent — prevents silently incorrect segmentations.

📏

Voxel Spacing Validation

Checks that spacing is within 0.5–5.0 mm per axis and anisotropy ratio < 3.0. Catches reconstruction artifacts.

🏃

Motion Artifact Scoring

Edge-energy analysis across the scan detects patient motion. Threshold-based warning before inference runs.

📐

Coverage Check

Ensures ≥ 20 slices for adequate anatomical coverage. Incomplete scans are flagged before reporting.

↕️

Slice Gap Detection

Identifies large inter-slice gaps that would compromise 3D volumetric accuracy.
